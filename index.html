Eksempel på løsning

Azure Landing Zone med tydelig separasjon mellom plattform og applikasjoner

Dette eksempelet viser hvordan en Azure-plattform kan bygges med standardiserte plattformtjenester for governance, sikkerhet, nettverk, kostnadskontroll og DevOps, samtidig som applikasjonene kan utvikles og driftes mer selvstendig.

Problem Mange miljøer blir unødig komplekse med for mange nivåer, for mange unntak og lite tydelig ansvar.
Løsning En standardisert plattform med policy, nettverk, sikkerhet, overvåkning og DevOps som felles fundament.
Effekt Lavere kompleksitet, bedre kontroll, enklere forvaltning og et bedre grunnlag for skalerbar drift.
